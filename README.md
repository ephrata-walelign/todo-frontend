# Todo App Project

## Project Description
This is a simple Todo app that allows users to add and delete tasks. The frontend is built with **HTML, CSS, and JavaScript**, and the backend uses **JSON Server** to provide a fake REST API.

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** JSON Server (fake REST API)
- **Deployment:** Netlify (frontend)
- **Documentation:** Swagger (backend API endpoints)

### Swagger Documentation

## Frontend Usage

1. Open the app in your browser:  
[Frontend Deployed URL](https://todo-final1.netlify.app/)

2. Add a new task:
   - Type the task in the input box
   - Click **Add**

3. Delete a task:
   - Click **Delete** button next to the task

---

## How to Run Locally

### Backend
1. Clone the backend repo:
   git clone https://github.com/ephrata-walelign/todo-backend.git
2. Install JSON Server (if not installed):
   npm install -g json-server
3. Run the backend:
   json-server --watch db.json --port 3000
